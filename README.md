# Automação de Controle de DIFAL (MG) 🚀

Este projeto foi desenvolvido para automatizar o processamento fiscal de arquivos XML de NF-e, focando no cálculo do DIFAL para o estado de Minas Gerais.

## 📋 Funcionalidades
- **Leitura Automática:** Processa todos os XMLs de uma pasta selecionada.
- **Identificação Fiscal:** Diferencia CFOPs de venda e remessa, além de identificar o CRT do fornecedor.
- **Cálculo Automático:** Aplica a alíquota de 7,35% para operações interestaduais destinadas a MG.
- **Relatório em Excel:** Gera uma planilha pronta com justificativas e tratativas fiscais.

## 🛠️ Tecnologias Utilizadas
- **Python 3.12**
- **Pandas** (Tratamento de dados)
- **Openpyxl** (Geração de Excel)
- **Tkinter** (Interface para seleção de pastas)
